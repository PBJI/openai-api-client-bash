# Installation
1) Download the runai script.
2) Do
> chmod +x runai
3) Do
> mv runai /bin/runai
4) Done
# Usage
1) For first time users do
> runai -a "openai_auth_key"
2) Press Ctrl+D two times
3) Press Ctrl + C
4) Restart the computer to load the changes
5) Run
> runai
6) Enter your prompt
7) Press Ctrl + D after entering the prompt each time
8) Wait for response
